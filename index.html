Основные теги в HTML

Теги в html делятся на два типа:

  1. Одинарные - состоят из одного тега,их не требуется закрывать.

  2. Парные - состоят из открывающего и закрывающего тега.

Для удобства одинарные теги будут обозначаться буквой "A", а двойные буквой "B".

  1. h1-h6 - теги используемые для создания заголовков.(A)

    2. При помщи него создана надпись "Основные теги в HTML" в самомо верху сайта

  2. a - используется для вставки ссылок в сайт.(B)

    3. На этом сайте он не используется.

  3. img - используется для вставки изображений в сайт(A)

    4. На этом сайте он не используется.

  4. p - используется для обозначения параграфа, или простыми словами создания текста.(A)

    5. При помощи него создана надпись про то что теги будут обозначаться буквой A и B.

  5. br - используется для создания отступов между строками.(B)

    6. При помощи него был сделан отступ межу описанием двойных и одинарных тегов

  6. div - используется для группирования элементов сайта.(A)

    7. Этот тег не будет виден и полезен посетителю сайта,он облегчает работу програмисту и делает код красивее.

  7. div - используется для создания линий

    8. При помощи него была сделана черная линия между содержимым и заголовком сайта.
